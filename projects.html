E-commerce Clothing Brand

I developed an online clothing brand from the ground up using HTML, CSS,
and JavaScript to create a dynamic and user-friendly shopping experience.
The entire site was built without the use of any editor or templates,
allowing for full customization. I integrated Stripe for secure and seamless
payment processing, ensuring a smooth checkout process for customers. The
website is fully responsive, offering an optimal experience across
devices. From product listings to the shopping cart, every feature was designed
with functionality and style in mind. The business generated revenue through
the sale of unique clothing items. This project showcases my skills in
front-end web development and payment integration.

Visit the site here: shopwestend.ca


Real Time Chat App

This project is a simple real-time chat application where multiple clients can
send and receive messages through a server. The server handles connections
and broadcasts messages to all clients, ensuring everyone stays updated.
Each client enters their name, and messages are displayed in the format
"Name - Message" for clarity. It demonstrates basic networking concepts using
sockets and threads in C to enable smooth communication.
